Put your dynamic category chart to any post or page

== Description ==

This simple plugin let's you to put your category chart to any post or page of your Wordpress site. It seems as JPG image, but it works dynamically via Google Chart API. Each time data is refreshed itself. 


== Installation ==

Simple:

1. Descompress the file
2. Upload the directory `category-chart/` to the `wp-content/plugins/` directory.
3. Activate the plugin
4. Go to Post/page editor, paste [category-chart] shorttag to any place of your post.

== Frequently Asked Questions ==

= How it works?=
